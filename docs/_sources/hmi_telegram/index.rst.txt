hmi_telegram
==================================================

.. raw:: html

    <p>hmi_telegram allows you to perform HMI queries to a user connected via Telegram.
  Because HMI queries define a Context Free Grammar, a list of completions can be suggested.
  These are presented to the user in a list of buttons to pick a next word from</p>
