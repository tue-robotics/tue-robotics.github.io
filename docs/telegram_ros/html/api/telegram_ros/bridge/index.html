_telegram_start_callback(update, _)[source]¶

Called when a Telegram user sends the ‘/start’ event to the bot, using this event, the bridge can be connected to a specific conversation. This checks whether the user is on the whitelist if the whitelist is a list and allows any user if whitelist is empty

Parameters:
