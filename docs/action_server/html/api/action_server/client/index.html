class action_server.client.Client(robot_name)[source]¶

A client for the action server

Wraps the client side of the actionlib interface so that it can be easily used in client side applications.

Example

client = Client(‘amigo’) semantics = “{‘actions’: [{‘action’: ‘say’, ‘sentence’: ‘ROBOT_NAME’}]}” client.send_task(semantics)

Send a task to the action server and return immediately. A task is composed of one or multiple actions.

Parameters:semantics – A json string with a list of dicts, every dict in the list has at least an ‘action’ field,

and depending on the type of action, several parameter fields may be required. :param done_cb: (callable) Callback that gets called on transitions to Done. The callback should take one parameter: TaskOutCome :param feedback_cb: (callable)Callback that gets called whenever feedback for this goal is received. Takes one parameter: the feedback.
