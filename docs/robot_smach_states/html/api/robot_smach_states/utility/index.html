class robot_smach_states.utility.WaitForTrigger(robot, triggers, topic, rate=1.0)

Bases: smach.State

This state will block execution until a suitable trigger command is received on the channel /trigger It will receive std_msgs.String and will compare it to the strings in the array that is given.

Example to wait for one of the strings ‘allow’ or ‘deny’ (could be sent from a gui):

WaitForTrigger(robot, [‘allow’, ‘deny’], “/trigger”),
transitions={ ‘allow’: ‘DO_SOMETHING’,

‘deny’: ‘DO_SOMETHING’, ‘preempted’: ‘failed’})
