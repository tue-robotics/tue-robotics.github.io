Smach state to find a person. The robot looks around and tries to recognize all faces in view.

Initialization method

Parameters:

  • robot – robot api object

  • person_label – (str) person label or a designator resolving to a str

  • search_timeout – (float) maximum time the robot is allowed to search

  • look_distance – (float) robot only considers laser entities within this radius

  • discard_other_labels – (bool) whether or not to discard recognitions based on the label

  • room – has to be the id of a room type in the knowledge (f.e. bedroom)

  • speak_when_found – bool indicating whether or not the robot should speak upon finding a person

  • look_range – from what to what head angle should the robot search (defaults to -90 to +90 deg)

  • look_steps – How many steps does it take in that range (default = 8)
