Initialization method

Parameters:

  • robot – robot api object

  • properties – (dict) (default: None) keyvalue pair of the properties a person must possess. None as a value for a property would search for all possible values of the property.

  • query_entity_designator – (default: None) An entity designator to match all found people to

  • found_people_designator – (default: None) A designator to write the search result to. The designator always has a list of found people written to it.

  • look_distance – (float) (default: 10.0) The distance (radius) which the robot must look at

  • speak – (bool) (default: False) If True, the robot will speak while trying to find a named person

  • strict –

    (bool) (default: True) Only used if properties is not None AND the {key:value} pair of a property has non None values. Disabled when setting reverse to True. If set to True then only people with all specified properties are returned, else all people with at least one true property. Example:

    properties = {‘tags’: [‘LWaving’, ‘RWaving’, ‘LHolding’, ‘RHolding’]} strict = True

    This will return a list of people who have the tags:

    ’LWaving’ AND ‘RWaving’ AND ‘LHolding’ AND ‘RHolding’

    strict = False
    This will return a list of people who have the tags:

    ’LWaving’ OR ‘RWaving’ OR ‘LHolding’ OR ‘RHolding’

  • reverse – (bool) (default: False) Want all the people that don’t match ANY property. So enabling reverse will make the property matching a not (x1 OR x2 OR …).

  • nearest – (bool) (default: False) If True, selects the people nearest to the robot who match the requirements posed using the properties, query_entity_designator, look distance and strict arguments

  • attempts – (int) (default: 1) Max number of search attempts

  • search_timeout – (float) (default: 60) maximum time the robot is allowed to search

  • look_range – (tuple of size 2) (default: (-np.pi/2, np.pi/2)) from what to what head angle should the robot search

  • look_steps – (int) (default: 8) How many steps does it take in that range
