class robot_smach_states.world_model.Inspect(robot, entityDes, objectIDsDes=None, searchArea='on_top_of', navigation_area='', unknown_threshold=0.0, filter_threshold=0.0)[source]¶

Bases: smach.StateMachine

Class to navigate to a(n) (furniture) object and segment the objects on top of it. Note that when inspecting a high entity the robot will end the Inspect in a high position.

Constructor

Parameters:
  • robot – robot object
  • entityDes – EdEntityDesignator indicating the (furniture) object to inspect
  • objectIDsDes – designator that is used to store the segmented objects
  • searchArea – string defining where the objects are w.r.t. the entity, default = on_top_of
  • navigation_area – string identifying the inspection area. If provided, NavigateToSymbolic is used. If left empty, NavigateToObserve is used.
  • unknown_threshold – Entities whose classification score is lower than this float are not marked with a type
  • filter_threshold – Entities whose classification score is lower than this float are ignored (i.e. are not added to the segmented_entity_ids_designator)
