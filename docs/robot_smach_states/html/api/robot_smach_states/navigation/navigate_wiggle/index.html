class robot_smach_states.navigation.navigate_wiggle.NavigateWiggle(robot, duration, vth=0.5, angle=0.15, speak=True)

Bases: smach.State

Wiggle the base of the robot. It rotates clockwise to :param: angle, followed by rotating anti-clockwise to minus :param: angle. This is repeated until it has to return to its original orientation. The decision is taken at each change of direction.

Parameters:

  • robot (robot_skills.robot.Robot) – Robot instance

  • duration (Union[rospy.Duration, float, int]) – Duration of the wiggle

  • vth (Union[float, int]) – Angular speed

  • angle (Union[float, int]) – Angle of the wiggle from the center

  • speak (bool) – Flag whether the robot should speak
