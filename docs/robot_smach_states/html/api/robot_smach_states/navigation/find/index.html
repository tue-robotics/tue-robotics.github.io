Find an entity based on a description. The description designator should have resolve type dict and it should contain at least a ‘type’ field

Parameters:

  • robot – Robot object

  • knowledge – Robocup knowledge object. Used to find an object by its properties

  • source_entity_designator – Designator resolving to EdEntity to inspect

  • description_designator – dict which contains at least a ‘type’ field

  • found_entity_designator – Designator the found entity can be written to

  • area_name – str or str Designator describing the area of the source entity to inspect

  • navigation_area – str or str Designator describing the area to drive to for the inspect. None=NavigateToObserve
