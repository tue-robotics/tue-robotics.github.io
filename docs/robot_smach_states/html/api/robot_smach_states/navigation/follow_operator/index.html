_register_operator()¶

Robots looks at the operator and asks whether the operator should follow. If he says yes, then set self._operator. Also adds the operator to the breadcrumb list

_update_breadcrumb_path()¶

If the last breadcrumb is less than a threshold away, replace the last breadcrumb with the latest operator position; otherwise just add it. In case a breadcrumb is ‘reached’, remove all breadcrumbs up to that point

_backup_register()¶

This only happens when the operator was just registered, and never tracked

_track_operator()¶

Sets self._operator_distance if we have an operator and otherwise set self._operator_distance to the distance to the last operator
