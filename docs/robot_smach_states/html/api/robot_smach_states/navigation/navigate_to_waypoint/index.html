Classes

NavigateToWaypoint

Move the robot to a specified location.

Module Contents

class robot_smach_states.navigation.navigate_to_waypoint.NavigateToWaypoint(robot, waypoint_designator, radius=0.15, look_at_designator=None, reset_head=True, speak=True, reset_pose=True)

Bases: robot_smach_states.navigation.navigation.NavigateTo

Move the robot to a specified location.

Parameters:

  • robot – Robot object

  • constraint_function – function resolving to a tuple(PositionConstraint, OrientationConstraint) telling the robot where to drive to.

  • reset_head – Whether or not the head should be used for obstacle avoidance during navigation.

  • speak – Whether or not the robot should speak during navigation

  • reset_pose – Whether or not the robot is allowed to change its pose for navigation.

Navigate to a waypoint in the world model

Parameters:

  • waypoint_designator – designator resolving to the waypoint

  • 0.15) (radius (default) – allowed distance to the waypoint

  • look_at_designator – Entity the robot should face from the waypoint. If not specified the robot will adopt the orientation of the waypoint

  • speak – Whether or not the robot should speak while driving.
