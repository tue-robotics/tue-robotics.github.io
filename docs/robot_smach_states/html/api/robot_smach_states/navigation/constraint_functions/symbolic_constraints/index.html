robot_smach_states.navigation.constraint_functions.symbolic_constraints.symbolic_constraint(robot, entity_designator_area_name_map)[source]¶

Generate navigation constraints based on an entity and an area of that entity.

Parameters:

  • robot – robot object

  • entity_designator_area_name_map – dictionary mapping EdEntityDesignators to a string or designator resolving to a string, representing the area, e.g., entity_designator_area_name_map[<EdEntity>] = ‘in_front_of’.

Returns:

navigation constraints. If one of the entities does not resolve, None is returned.

Return type:

tuple(PositionConstraint, OrientationConstraint)
