Navigates to a radius from a waypoint

Parameters:
  • waypoint_designator – designator resolving to the waypoint
  • radius – allowed distance to the waypoint
  • look – whether or not to take the orientation of the waypoint (default True)
Returns:

navigation constraints, if the entity does not resolve, None is returned.
Return type:

tuple(PositionConstraint, OrientationConstraint)
