Generate navigation constraints for a radius from a pose

Parameters:

  • y (x,) – x and y coordinates

  • rz – orientation to assume. None=no orientation constraint (default None)

  • radius – allowed distance to the pose

  • frame_id – frame in which the pose is expressed

Returns:

navigation constraints

tuple(PositionConstraint, OrientationConstraint)
