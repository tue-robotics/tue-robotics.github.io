class robot_smach_states.manipulation.manipulation.HandoverFromHuman(robot, arm_designator, grabbed_entity_label='', grabbed_entity_designator=None, timeout=15, arm_configuration='handover_to_human')

Bases: smach.StateMachine

State that enables low level grab reflex. Besides a robot object, needs an arm and an entity to grab, which is either one from ed through the grabbed_entity_designator or it is made up in the CloseGripperOnHandoverToRobot state and given the grabbed_entity_label as id.

Hold up hand to accept an object and close hand once something is inserted

Parameters:

  • robot – Robot with which to execute this behavior

  • arm_designator – ArmDesignator resolving to arm accept item into

  • grabbed_entity_label – What ID to give a dummy item in case no grabbed_entity_designator is supplied

  • grabbed_entity_designator – EntityDesignator resolving to the accepted item. Can be a dummy

  • timeout – How long to hold hand over before closing without anything

  • arm_configuration – Which pose to put arm in when holding hand up for the item.
