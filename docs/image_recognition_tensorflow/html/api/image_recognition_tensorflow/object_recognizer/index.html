class image_recognition_tensorflow.object_recognizer.ObjectRecognizer(graph_path, labels_path, input_tensor, output_tensor)

Bases: object

Recognize objects via a given TensorFlow pre-trained neural net. The configuration oof the in/output tensor names are preset to those of a (retrained) Inception_v3 network

Parameters:

  • graph_path – path to a pre-trained TensorFlow graph.pb file, e.g. output_graph.pb

  • labels_path – path to a file containing labels. One label per line, order should match that of the graph

  • input_tensor – Which tensor to feed the numpy-formatted image in?

  • output_tensor – Which tensor to extract the result from?
