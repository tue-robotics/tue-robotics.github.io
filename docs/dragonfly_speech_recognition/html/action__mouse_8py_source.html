22 Mouse action
23 ============================================================================
24 
25 This section describes the :class:`Mouse` action object. This type of
26 action is used for controlling the mouse cursor and clicking mouse
27 button.
28 
29 Below you'll find some simple examples of :class:`Mouse` usage, followed
30 by a detailed description of the available mouse events.
31 
32 
33 Example mouse actions
34 ............................................................................
35 
36 The following code moves the mouse cursor to the center of the foreground
37 window (``(0.5, 0.5)``) and then clicks the left mouse button once
38 (``left``)::
39 
40  # Parentheses ("(...)") give foreground-window-relative locations.
41  # Fractional locations ("0.5", "0.9") denote a location relative to
42  # the window or desktop, where "0.0, 0.0" is the top-left corner
43  # and "1.0, 1.0" is the bottom-right corner.
44  action = Mouse("(0.5, 0.5), left")
45  action.execute()
46 
47 The line below moves the mouse cursor to 100 pixels left of the primary
48 monitor's left edge (if possible) and 250 pixels down from its top edge
49 (``[-100, 250]``), and then double clicks the right mouse button
50 (``right:2``)::
51 
52  # Square brackets ("[...]") give desktop-relative locations.
53  # Integer locations ("1", "100", etc.) denote numbers of pixels.
54  # Negative numbers ("-100") are counted from the left-edge of the
55  # primary monitor. They are used to access monitors above or to the
56  # left of the primary monitor.
57  Mouse("[-100, 250], right:2").execute()
58 
59 The following command drags the mouse from the top right corner of the
60 foreground window (``(0.9, 10), left:down``) to the bottom left corner
61 (``(25, -0.1), left:up``)::
62 
63  Mouse("(0.9, 10), left:down, (25, -0.1), left:up").execute()
64 
65 The code below moves the mouse cursor 25 pixels right and 25 pixels up
66 (``<25, -25>``)::
67 
68  # Angle brackets ("<...>") move the cursor from its current position
69  # by the given number of pixels.
70  Mouse("<25, -25>").execute()
71 
72 
73 Mouse specification format
74 ............................................................................
75 
76 The *spec* argument passed to the :class:`Mouse` constructor specifies
77 which mouse events will be emulated. It is a string consisting of one or
78 more comma-separated elements. Each of these elements has one of the
79 following possible formats:
80 
81 Mouse movement actions:
82 
83  - move the cursor relative to the top-left corner of the desktop monitor
84  containing coordinates ``[0, 0]`` (i.e. the primary monitor):
85  ``[`` *number* ``,`` *number* ``]``
86  - move the cursor relative to the foreground window:
87  ``(`` *number* ``,`` *number* ``)``
88  - move the cursor relative to its current position:
89  ``<`` *pixels* ``,`` *pixels* ``>``
90 
91 In the above specifications, the *number* and *pixels* have the
92 following meanings:
93 
94  - *number* -- can specify a number of pixels or a fraction of
95  the reference window or desktop. For example:
96 
97  - ``(10, 10)`` -- 10 pixels to the right and down from the
98  foreground window's left-top corner
99  - ``(0.5, 0.5)`` -- center of the foreground window
100 
101  - *pixels* -- specifies the number of pixels
102 
103 Mouse button-press action:
104  *keyname* [``:`` *repeat*] [``/`` *pause*]
105 
106  - *keyname* -- Specifies which mouse button to click:
107 
108  - ``left`` -- left mouse button key
109  - ``middle`` -- middle mouse button key
110  - ``right`` -- right mouse button key
111  - ``four`` -- fourth mouse button key
112  - ``five`` -- fifth mouse button key
113  - ``wheelup`` -- mouse wheel up
114  - ``stepup`` -- mouse wheel up 1/3
115  - ``wheeldown`` -- mouse wheel down
116  - ``stepdown`` -- mouse wheel down 1/3
117  - ``wheelright`` -- mouse wheel right
118  - ``stepright`` -- mouse wheel right 1/3
119  - ``wheelleft`` -- mouse wheel left
120  - ``stepleft`` -- mouse wheel left 1/3
121 
122  - *repeat* -- Specifies how many times the button should be clicked:
123 
124  - ``0`` -- don't click the button, this is a no-op
125  - ``1`` -- normal button click
126  - ``2`` -- double-click
127  - ``3`` -- triple-click
128 
129  - *pause* --
130  Specifies how long to pause *after* clicking the button. The value
131  should be an integer giving in hundredths of a second. For example,
132  ``/100`` would mean one second, and ``/50`` half a second.
133 
134 Mouse button-hold or button-release action:
135  *keyname* ``:`` *hold-or-release* [``/`` *pause*]
136 
137  - *keyname* -- Specifies which mouse button to click; same as above.
138 
139  - *hold-or-release* --
140  Specified whether the button will be held down or released:
141 
142  - ``down`` -- hold the button down
143  - ``up`` -- release the button
144 
145  - *pause* --
146  Specifies how long to pause *after* clicking the button; same as above.
147 
148 
149 Mouse across platforms
150 ............................................................................
151 
152 Please note that there are some platforms which do not support emulating
153 every mouse button listed above. If an unsupported mouse button (*keyname*)
154 is specified and the :class:`Mouse` action executed, an error is raised. For
155 instance, scrolling the mouse wheel horizontally (e.g. *wheelleft*) is not,
156 by default, a supported operation on X11::
157 
158  ValueError: Unsupported scroll event: wheelleft
159 
160 Fortunately, this particular problem can be fixed by installing the *pynput*
161 library::
162 
163  pip install pynput
164 
165 On MacOS, however, Dragonfly cannot be used to scroll horizontally.

202  def _parse_spec(self, spec):
203  """ Convert the given *spec* to keyboard events. """
204  events = []
205  parts = self._split_parts(spec)
206  for part in parts:
207  handled = False
208  for handler in self._handlers:
209  try:
210  if handler(self, part, events):
211  handled = True
212  break
213  except Exception:
214  continue
215  if not handled:
216  raise ActionError("Invalid mouse spec: %r (in %r)"
217  % (part, spec))
218  return events

220  def _parse_position_pair(self, spec):
221  parts = spec.split(",")
222  if len(parts) != 2:
223  raise ValueError("Invalid position pair spec: %r" % spec)
224 
225  h_origin, h_value = self._parse_position(parts[0])
226  v_origin, v_value = self._parse_position(parts[1])
227 
228  return (h_origin, h_value, v_origin, v_value)

230  def _process_window_position(self, spec, events):
231  if not spec.startswith("(") or not spec.endswith(")"):
232  return False
233  h_origin, h_value, v_origin, v_value = self._parse_position_pair(spec[1:-1])
234  event = MoveWindowEvent(h_origin, h_value, v_origin, v_value)
235  events.append(event)
236  return True
237 
238  def _process_screen_position(self, spec, events):
239  if not spec.startswith("[") or not spec.endswith("]"):
240  return False
241  _, h_value, _, v_value = self._parse_position_pair(spec[1:-1])
242  event = MoveScreenEvent(True, h_value, True, v_value)
243  events.append(event)
244  return True
245 
246  def _process_relative_position(self, spec, events):
247  if not spec.startswith("<") or not spec.endswith(">"):
248  return False
249  parts = spec[1:-1].split(",")
250  if len(parts) != 2:
251  return False
252  horizontal = int(parts[0])
253  vertical = int(parts[1])
254  event = MoveRelativeEvent(horizontal, vertical)
255  events.append(event)
256  return True

261  def _process_button(self, spec, events):
262  parts = spec.split(":", 1)
263  button = parts[0].strip()
264  if len(parts) == 1: special = 1
265  else: special = parts[1].strip()
266 
267  if button in self._button_flags:
268  flag_down, flag_up = self._button_flags[button]
269 
270  if special == "down":
271  event = ButtonEvent(flag_down)
272  elif special == "up":
273  event = ButtonEvent(flag_up)
274  else:
275  try:
276  repeat = int(special)
277  except ValueError:
278  return False
279  flag_series = (flag_down, flag_up) * repeat
280  event = ButtonEvent(*flag_series)
281  elif button in self._wheel_flags:
282  flag = self._wheel_flags[button]
283  try:
284  repeat = int(special)
285  except ValueError:
286  return False
287  flag = (flag[0], repeat * flag[1])
288  event = ButtonEvent(flag)
289  else:
290  return False
291 
292  events.append(event)
293  return True

295  def _process_pause(self, spec, events):
296  if not spec.startswith("/"):
297  return False
298  interval = float(spec[1:]) / 100
299  event = PauseEvent(interval)

301  return True

311  def _parse_position(self, spec):
312  spec = spec.strip()
313 
314  if spec.startswith("-"): origin = False
315  else: origin = True
316 
317  if spec.find(".") != -1: value = float(spec)
318  else: value = int(spec)
319 
320  return (origin, value)

322  def _split_parts(self, spec):
323 
324  delimiters = ["()", "[]", "<>"]
325 
326  parts = spec.split(",")
327  items = []
328  while parts:
329  part = parts.pop(0).strip()
330 
331  found_delimiter = False
332  for begin, end in delimiters:
333  if begin not in part:
334  continue
335 
336  item = [part]
337  while parts:
338  part = parts.pop(0)
339  item.append(part)
340  if end in part:
341  break
342  if end not in part:
343  raise Exception("No closing delimiter found for %r" % item[0])
344 
345  new_item = ",".join(item)
346  found_delimiter = True
347  break
348 
349  if not found_delimiter:
350  new_item = part
351 
352  if "/" in new_item:
353  before, after = new_item.split("/", 1)
354  items.append(before.strip())
355  items.append("/" + after.strip())
356  else:
357  items.append(new_item)
358 
359  return items
