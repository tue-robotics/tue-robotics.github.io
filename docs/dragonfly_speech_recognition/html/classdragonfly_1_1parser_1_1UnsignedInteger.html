Detailed Description

    Parser element for unsigned integer literals.

    Simple usage examples:
    >>> from dragonfly.parser import Parser
    >>> parser = Parser(UnsignedInteger())
    >>> parser.parse("0")
    0
    >>> parser.parse("1234")
    1234
    >>> parser.parse("0001234")
    1234
