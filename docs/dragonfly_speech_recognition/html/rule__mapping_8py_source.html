22 MappingRule class
23 ============================================================================
24 
25 The MappingRule class is designed to make it very easy to create a rule
26 based on a mapping of spoken-forms to semantic values.
27 
28 This class has the following parameters to customize its behavior:
29 
30  - *mapping* -- mapping of spoken-forms to semantic values
31  - *extras* -- extras elements referenced from the compound spec
32  - *defaults* -- default values for the extras
33  - *exported* -- whether the rule is exported
34  - *context* -- context in which the rule will be active
35 
36 Each of these parameters can be passed as a (keyword) arguments to the
37 constructor, or defined as a class attribute in a derived class.
38 
39 
40 Example usage
41 ............................................................................
42 
43 The MappingRule class can be used to define a voice-command as follows::
44 
45  class ExampleRule(MappingRule):
46 
47  mapping = {
48  "[feed] address [bar]": Key("a-d"),
49  "subscribe [[to] [this] feed]": Key("a-u"),
50  "paste [feed] address": Key("a-d, c-v, enter"),
51  "feeds | feed (list | window | win)": Key("a-d, tab:2, s-tab"),
52  "down [<n>] (feed | feeds)": Key("a-d, tab:2, s-tab, down:%(n)d"),
53  "up [<n>] (feed | feeds)": Key("a-d, tab:2, s-tab, up:%(n)d"),
54  "open [item]": Key("a-d, tab:2, c-s"),
55  "newer [<n>]": Key("a-d, tab:2, up:%(n)d"),
56  "older [<n>]": Key("a-d, tab:2, down:%(n)d"),
57  "mark all [as] read": Key("cs-r"),
58  "mark all [as] unread": Key("cs-u"),
59  "search [bar]": Key("a-s"),
60  "search [for] <text>": Key("a-s") + Text("%(text)s\\n"),
61  }
62  extras = [
63  Integer("n", 1, 20),
64  Dictation("text"),
65  ]
66  defaults = {
67  "n": 1,
68  }
69 
70  rule = ExampleRule()
71  grammar.add_rule(rule)
