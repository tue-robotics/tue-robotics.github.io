Detailed Description

    Parser element for quoted strings.

    Simple usage examples:
    >>> from dragonfly.parser import Parser
    >>> parser = Parser(Integer())
    >>> parser.parse("0")
    0
    >>> parser.parse("+0")
    0
    >>> parser.parse("-000")
    0
    >>> parser.parse("1234")
    1234
    >>> parser.parse("+001234")
    1234
    >>> parser.parse("-001234")
    -1234
