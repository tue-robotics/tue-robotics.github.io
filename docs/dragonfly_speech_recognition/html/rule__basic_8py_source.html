22 BasicRule class
23 ============================================================================
24 
25 The BasicRule class is designed to make it easy to create a rule from an
26 element tree, rather than building one indirectly via compound element
27 specs.
28 
29 This rule class has the following parameters to customize its behavior:
30 
31  - *name* (*str*) -- the rule's name
32  - *element* (*Element*) -- root element for this rule
33  - *exported* -- whether the rule is exported
34  - *context* -- context in which the rule will be active
35 
36 Each of these parameters can be passed as a (keyword) arguments to the
37 constructor, or defined as a class attribute in a derived class.
38 
39 .. note::
40 
41  The BasicRule class has only limited support for the "extras" and
42  "defaults" functionality of the
43  :class:`dragonfly.grammar.rule_compound.CompoundRule` and
44  :class:`dragonfly.grammar.rule_mapping.MappingRule` classes. By default,
45  the *extras* dictionary passed to :meth:`_process_recognition` will
46  only contain an entry for the root element of the rule.
47 
48 
49 Example usage
50 ............................................................................
51 
52 The BasicRule class can be used to define a voice-command as follows::
53 
54  from dragonfly import (BasicRule, Repetition, Alternative, Literal, Text,
55  Grammar)
56 
57  class ExampleRule(BasicRule):
58  # Define a rule element that accepts 1 to 5 (exclusive) repetitions
59  # of either 'test one', 'test two' or 'test three'. These commands
60  # type their respective numbers in succession using the Text action.
61  element = Repetition(
62  Alternative((
63  Literal("test one", value=Text("1")),
64  Literal("test two", value=Text("2")),
65  Literal("test three", value=Text("3")),
66  )),
67  1, 5
68  )
69 
70  # Create a grammar with the example rule and load it.
71  rule = ExampleRule()
72  grammar = Grammar("BasicRule Example")
73  grammar.add_rule(rule)
74  grammar.load()
75 
76 
77 The above :class:`BasicRule` example can be defined without sub-classing::
78 
79  rule = BasicRule(
80  element=Repetition(
81  Alternative((
82  Literal("test one", value=Text("1")),
83  Literal("test two", value=Text("2")),
84  Literal("test three", value=Text("3")),
85  )),
86  1, 5)
87  )

203  This is the method which should be overridden in most cases
