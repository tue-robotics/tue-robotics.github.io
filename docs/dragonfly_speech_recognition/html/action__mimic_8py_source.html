22 Mimic action
23 ============================================================================
24 
25 
26 The :class:`Mimic` action mimics a single recognition. This is useful for
27 repeating a single prerecorded or predefined voice-command.
28 
29 This class could for example be used to open a new Windows Explorer window::
30 
31  action = Mimic("open", "windows", "explorer")
32  action.execute()
33 
34 
35 A more in-depth example is given below in the class reference.
36 
37 
38 Mimic quirks
39 ----------------------------------------------------------------------------
40 
41 Some SR engine back-ends have confusing :meth:`engine.mimic` method
42 behavior. See the engine-specific mimic method documentation in sections
43 under :ref:`RefEngines` for more information.
44 
45 
46 Class reference
47 ----------------------------------------------------------------------------
48 
49 """

51 from six import string_types

53 from dragonfly.actions.action_base import ActionBase, ActionError
54 from dragonfly.engines import get_engine

57 #---------------------------------------------------------------------------
58 
59 class Mimic(ActionBase):
60  """
61  Mimic recognition action.
62 
63  The constructor arguments are the words which will be mimicked.
64  These should be passed as a variable argument list. For example::
65 
66  action = Mimic("hello", "world", r"!\\exclamation-mark")
67  action.execute()
68 
69  If an error occurs during mimicking the given recognition, then an
70  *ActionError* is raised. A common error is that the engine does
71  not know the given words and can therefore not recognize them.
72  For example, the following attempts to mimic recognition of *one
73  single word* including a space and an exclamation-mark; this will
74  almost certainly fail::
75 
76  Mimic("hello world!").execute() # Will raise ActionError.
77 
78  The constructor accepts the optional *extra* keyword argument, and
79  uses this to retrieve dynamic data from the extras associated with
80  the recognition. For example, this can be used as follows to
81  implement dynamic mimicking::
82 
83  class ExampleRule(MappingRule):
84  mapping = {
85  "mimic recognition <text> [<n> times]":
86  Mimic(extra="text") * Repeat(extra="n"),
87  }
88  extras = [
89  IntegerRef("n", 1, 10),
90  Dictation("text"),
91  ]
92  defaults = {
93  "n": 1,
94  }
95 
96  The example above will allow the user to speak **"mimic
97  recognition hello world! 3 times"**, which would result in the
98  exact same output as if the user had spoken **"hello world!"**
99  three times in a row.
100 
101  """
102 
103  def __init__(self, *words, **kwargs):
104  ActionBase.__init__(self)
105  self._words = tuple(words)
106  if "extra" in kwargs:
107  self._extra = kwargs.pop("extra")
108  else:
109  self._extra = None
110 
111  # Set pretty printing string used by __str__ and __unicode__.
112  self._str = u", ".join(repr(w) for w in self._words)
113 
114  # Make sure that all keyword arguments have been consumed.
115  if kwargs:
116  raise ActionError("Invalid arguments: %r"
117  % ", ".join(list(kwargs.keys())))
118 
119  def _execute(self, data=None):
120  engine = get_engine()
121  words = self._words
122 
123  # If an extra was given, retrieve the associated value from
124  # the *data* dict and append it to the static words.
125  if self._extra:
126  try:
127  extra = data[self._extra]
128  except KeyError:
129  raise ActionError("No extra data available for extra %r"
130  % self._extra)
131 
132  # Append the extra data to the static words depending on
133  # the type of the extra data object.
134  if isinstance(extra, engine.DictationContainer):
135  words += tuple(extra.words)
136  elif isinstance(extra, (tuple, list)):
137  words += tuple(extra)
138  elif isinstance(extra, string_types):
139  words += (extra,)
140  else:
141  raise ActionError("Invalid extra data type: %r" % extra)
142 
143  # Mimic the series of words.
144  self._log.debug("Mimicking recognition: %r", words)
145  try:
146  engine.disable_recognition_observers()
147  engine.mimic(words)
148  engine.enable_recognition_observers()
149  except Exception as e:
150  raise ActionError("Mimicking failed: %s" % e)
