36 class WaitWindow(ActionBase):
37  """
38  Wait for a specific window context action.
39 
40  Constructor arguments:
41  - *title* (*str*) --
42  part of the window title: not case sensitive
43  - *executable* (*str*) --
44  part of the file name of the executable; not case sensitive
45  - *timeout* (*int* or *float*) --
46  the maximum number of seconds to wait for the correct
47  context, after which an :class:`ActionError` will
48  be raised.
49 
50  When this action is executed, it waits until the correct window
51  context is present. This window context is specified by the
52  desired window title of the foreground window and/or the
53  executable name of the foreground application. These are
54  specified using the constructor arguments listed above. The
55  substring search used is *not* case sensitive.
56 
57  If the correct window context is not found within *timeout*
58  seconds, then this action will raise an :class:`ActionError` to
59  indicate the timeout.
