37 
38  - :meth:`process_buffer`
39  - :meth:`process_wave_file`
40  - :meth:`recognise_forever`
41 
42  Timer functions will run whether or not recognition is paused
43  (i.e. in sleep mode).
44 
45  **Note**: long-running timers will block dragonfly from processing what
46  was said, so be careful with how you use them! Audio frames will not
47  normally be dropped because of timers, long-running or otherwise.
48 
49  Normal threads can be used instead of timers if desirable. This is
