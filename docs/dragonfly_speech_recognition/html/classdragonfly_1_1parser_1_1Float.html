Detailed Description

    Parser element for decimal fraction literals.

    Usage examples:
    >>> from dragonfly.parser import Parser
    >>> parser = Parser(Float())
    >>> parser.parse("0.0")
    0.0
    >>> parser.parse(".000")
    0.0
    >>> parser.parse("-.0")
    0.0
    >>> parser.parse("1.0")
    1.0
    >>> parser.parse("-1.0")
    -1.0
    >>> parser.parse("-1.75")
    -1.75
