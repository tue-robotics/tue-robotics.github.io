22 PlaySound action
23 ============================================================================
24 
25 The :class:`PlaySound` action class is used to play wave files.
26 
27 
28 Example usage
29 ----------------------------------------------------------------------------
30 
31 The following example shows how to play a wave file using the
32 :class:`PlaySound` action class::
33 
34  PlaySound(file="tada.wav").execute()
35 
36 
37 Windows
38 ----------------------------------------------------------------------------
39 
40 On Windows, :class:`PlaySound` uses the `PlaySound Windows API function
41 <https://docs.microsoft.com/en-us/windows/win32/multimedia/the-playsound-function>`__.
42 
43 The action can be used to play Windows system sounds. For example::
44 
45  # Play the system shutdown sound.
46  PlaySound("SystemExit").execute()
47 
48  # Play the logout sound.
49  PlaySound("WindowsLogout").execute()
50 
51 
52 System sound names are matched against registry keys.
53 
54 Invalid file paths or unknown system sounds will result in the default error
55 sound being played. ``RuntimeErrors`` will be raised if Windows fails to
56 play a known system sound.
57 
58 
59 Other platforms
60 ----------------------------------------------------------------------------
61 
62 On other platforms, the :class:`PlaySound` class will use `PyAudio
63 <http://people.csail.mit.edu/hubert/pyaudio>`__ to play specified wave
64 files.
65 
66 Invalid file paths will result in errors on other platforms.
67

100 def _get_pa_instance():
101  # Suppress initial ALSA messages if using ALSA.
102  # Got this from: https://stackoverflow.com/a/17673011/12157649
103  try:
104  asound = cdll.LoadLibrary('libasound.so')
105  c_error_handler = ERROR_HANDLER_FUNC(
106  lambda filename, line, function, err, fmt: None
107  )
108  asound.snd_lib_error_set_handler(c_error_handler)
109  except:
110  # We'll most likely get here if the Port Audio host API isn't ALSA.
111  asound = None
112 
113  # Create the pa instance.
114  pa = pyaudio.PyAudio()
115 
116  # If necessary, restore the original error handler.
117  if asound:
118  asound.snd_lib_error_set_handler(None)

122 def _pyaudio_play(name):
123  if not name:
124  return
125 
126  if pyaudio is None:
127  # Raise an error because pyaudio isn't installed.
128  raise RuntimeError("pyaudio must be installed to use PlaySound "
129  "on this platform")
130 
131  # Play the wave file.
132  pa = _get_pa_instance()
133  chunk = 1024
134  wf = wave.open(name, 'rb')
135  stream = pa.open(format=pa.get_format_from_width(wf.getsampwidth()),
136  channels=wf.getnchannels(), rate=wf.getframerate(),
137  output=True)
138  data = wf.readframes(chunk)
139  while data:
140  stream.write(data)
141  data = wf.readframes(chunk)
142 
143  stream.stop_stream()
144  stream.close()
145  pa.terminate()
