Create a matrix of the one-hot ground truths for each classification :return: numpy.ndarray of shape (number of classifications, number of labels) Axis 0 indexes over classifications, axis 1 is the score for the class at that position (i.e. 1 because this takes the ground truth as input) result[x][y] is 1 when the ground truth class for classification x is labels[y]. The other values are 0
