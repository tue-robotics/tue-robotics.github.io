ed_object_models.conversion_sdf.unique_name(name, names)[source]

Provide a unique name based on name. If name already in names, the name is changed. If the name ends with digits, this number is increased. If the name doesn’t end with a digit, the name is append with ‘1’. This continues until a unique name is found.

Parameters:

  • name (str) – name you want to be unique

  • names (List[str]) – list of names which are already used
