.. function:: euler_z_to_quaternion(angle)

   Calculate a quaternion based on a Z angle

   >>> from math import pi
   >>> euler_z_to_quaternion(pi/2)
   x: 0.0
   y: 0.0
   z: 0.707106781187
   w: 0.707106781187


.. function:: euler_z_from_quaternion(quaternion)

   Return the yaw (z) angle of a quaternion

   >>> from geometry_msgs.msg import Quaternion
   >>> q = Quaternion()
   >>> euler_z_from_quaternion(q)
   0.0
