property distance¶

Returns the measured distance. If the measurement is too old or the distance is outside of of the provided limits, NaN is returned

Returns:

float with distance if valid, NaN otherwise

property is_holding¶

Returns if the gripper is holding anything based on the measurement, i.e., if the measurement is recent and the value is between the lower limit and the sensor threshold

Returns:

bool if holding

property is_unknown¶

Returns if the state is unknown, i.e., either the measurement is outdated or the distance is less than the limit

Returns:

bool if unknown

property is_empty¶

Returns if the gripper is empty, i.e., the measurement is recent and the value is greater than the threshold

Returns:

bool if holding
