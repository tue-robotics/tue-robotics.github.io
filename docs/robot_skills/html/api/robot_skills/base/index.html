force_drive(self, vx, vy, vth, timeout, loop_rate=10, stop=True, ax=float('inf'), ay=float('inf'), ath=float('inf'))[source]¶

Forces the robot to drive by sending a command velocity directly to the base controller. N.B.: all collision avoidance is bypassed.

Parameters:
  • vx – forward velocity [m/s]
  • vy – sideways velocity [m/s]
  • vth – rotational velocity in [rad/s]
  • timeout – duration for this motion [s]
  • loop_rate – Rate of sending twist messages [Hz]
  • stop – send stop message afterwards
  • ax – forward acceleration [m/s^2]
  • ay – sideways acceleration [m/s^2]
  • ath – rotational acceleration [rad/s^2]
